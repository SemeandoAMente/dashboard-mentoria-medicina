# 🚀 GUIA DE DEPLOY - VERCEL (100% GRÁTIS)

## ⚡ PASSO A PASSO RÁPIDO

### 1️⃣ CRIAR CONTA NA VERCEL (1 minuto)

1. Acesse: https://vercel.com
2. Clique em **"Sign Up"**
3. Escolha **"Continue with GitHub"** (ou Email se preferir)
4. Autorize a Vercel a acessar o GitHub

---

### 2️⃣ SUBIR O PROJETO NO GITHUB (3 minutos)

**OPÇÃO A - Usando GitHub Desktop (Mais Fácil)**

1. Baixe o GitHub Desktop: https://desktop.github.com
2. Instale e faça login com sua conta GitHub
3. Clique em **"Add"** → **"Add Existing Repository"**
4. Selecione a pasta do dashboard
5. Clique em **"Publish Repository"**
6. Deixe o nome como **"dashboard-mentoria-medicina"**
7. **IMPORTANTE**: Desmarque "Keep this code private" se quiser público
8. Clique em **"Publish Repository"**

**OPÇÃO B - Usando Linha de Comando (Git)**

Abra o terminal/cmd na pasta do dashboard e execute:

```bash
git init
git add .
git commit -m "Dashboard Mentoria Medicina - Primeira versão"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/dashboard-mentoria-medicina.git
git push -u origin main
```

*(Substitua SEU-USUARIO pelo seu username do GitHub)*

---

### 3️⃣ FAZER DEPLOY NA VERCEL (2 minutos)

1. Acesse: https://vercel.com/new
2. Você verá a lista dos seus repositórios do GitHub
3. Procure por **"dashboard-mentoria-medicina"**
4. Clique em **"Import"**
5. **NÃO MUDE NADA** nas configurações
6. Clique em **"Deploy"**
7. Aguarde 30-60 segundos
8. **PRONTO!** 🎉

---

### 4️⃣ PEGAR O LINK FINAL

Após o deploy:

1. A Vercel mostrará uma tela de sucesso
2. Você verá um link tipo: **`https://dashboard-mentoria-medicina.vercel.app`**
3. **COPIE ESSE LINK**
4. Envie para sua aluna!

## ❓ PROBLEMAS?

**Erro 404?**
- Verifique se o arquivo `index.html` está na raiz da pasta

**Site não carrega?**
- Limpe o cache do navegador (Ctrl + Shift + R)
- Aguarde 2-3 minutos após o deploy

**Precisa atualizar o site?**
1. Faça as mudanças nos arquivos
2. Use o GitHub Desktop para fazer commit
3. Clique em "Push origin"
4. A Vercel atualiza AUTOMATICAMENTE em 30 segundos!
