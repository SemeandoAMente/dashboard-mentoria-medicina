# 🩺 Dashboard Mentoria Medicina

Dashboard visual de acompanhamento para cronograma de estudos de 12 semanas para vestibular de Medicina.

## 📋 Características

- **12 semanas de cronograma estruturado**
- **Técnica Espelho**: Segunda ↔ Quinta, Terça ↔ Sexta, Quarta ↔ Sábado
- **Técnica do Acúmulo**: Revisão acumulativa toda segunda-feira
- **Português diário** por questões (paralelo fixo)
- **1 discursiva por dia** vinculada à matéria âncora
- **Princípio 80/20**: Foco nos assuntos de maior incidência

## 🎯 Estrutura das Matérias

### Âncoras (Prioridade Máxima)
- Biologia
- Química
- Física

### Apoio Forte (Prioridade Alta)
- Matemática

### Rotativas (Prioridade Média)
- História
- Geografia
- Língua Estrangeira

## 🗓️ Grade Semanal

- **Segunda/Quinta**: Biologia + Matemática (Discursiva: Biologia)
- **Terça/Sexta**: Química + História/Geografia (Discursiva: Química)
- **Quarta/Sábado**: Física + Rotativa (Discursiva: Física)
- **Domingo**: Folga

## 🚀 Como Usar

1. Abra o `index.html` em qualquer navegador
2. Navegue pelas seções usando o menu lateral
3. Marque seu progresso conforme avança no cronograma
4. O progresso é salvo automaticamente no navegador

## 📱 Responsivo

Funciona perfeitamente em desktop, tablet e celular.
